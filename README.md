# catfish

The catfish component provides a daemon, manager, and editor to
centralize the configuration management of the Xfce system.

### Installation

From source: 

    % cd catfish
    % ./autogen.sh
    % make
    % make install

From release tarball:

    % tar xf catfish-<version>.tar.bz2
    % cd catfish-<version>
    % ./configure
    % make
    % make install
